So, what is bootstrap?

Getbootstrap.com was invented by Mark Otto (pictured left) and is currently run by a small team of developers on github. Bootstrap is a plugin used by coders and web developers who use HTML, CSS and JS. It's used to quickly develop and prototype fully responsive, mobile-first websites and apps.

Bootstrap uses automatic built in components, such as a responsive grid system and sass variables. It's very easy to set up and makes web design and coding a lot less time consuming!

In order to use bootstrap go to: www.getboostrap.com. To find the starter template, just click "Get Started" and scroll down. Copy and paste that into your text editor and your good to go! You can browse bootstrap components and features in the documentation tab.
